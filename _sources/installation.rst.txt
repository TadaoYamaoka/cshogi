Installation
============

Installation from GitHub Source
-------------------------------

Install with the following command. Cython and a compatible C++ compiler are required.

::

    pip install git+https://github.com/TadaoYamaoka/cshogi

Installation from PYPI
----------------------

::

    pip install cshogi

You need pip version 19.0 or higher. If you have a version lower than 19.0, you need to upgrade pip `first <https://pip.pypa.io/en/stable/installing/#upgrading-pip>`_.
